# Fonts Directory

Place your Sakkal Majalla font files here:

- `SakkalMajalla.ttf` or `SakkalMajalla.woff` (regular)
- `SakkalMajalla-Bold.ttf` or `SakkalMajalla-Bold.woff` (bold)

Supported formats: `.ttf`, `.woff`, `.woff2`, `.eot`

After placing the font files, the CSS will automatically load them.
